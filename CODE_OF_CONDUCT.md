# Code of Conduct

## Our commitment

This project helps people experiencing domestic violence. Everyone contributing to it must treat others with respect and empathy.

## Standards

**Expected behavior:**
- Be kind and respectful
- Accept constructive feedback
- Focus on what is best for the people this platform serves
- Respect the sensitivity of the subject matter

**Unacceptable behavior:**
- Harassment, trolling, or personal attacks
- Publishing others' private information
- Dismissive or trivializing language about domestic violence
- Any conduct that would be inappropriate given the project's purpose

## Enforcement

Instances of unacceptable behavior can be reported to info@nkatrori.org. All complaints will be reviewed. Maintainers have the right to remove contributions and ban contributors who violate this code.

## Scope

This applies to all project spaces: issues, pull requests, discussions, and any public space where someone represents this project.
